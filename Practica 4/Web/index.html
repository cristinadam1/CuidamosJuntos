Introducción

El objetivo de esta aplicación es facilitar la organización de actividades para residencias de ancianos, fomentando la colaboración entre voluntarios externos, residentes y sus familiares. La app está diseñada para atender las necesidades de cuatro tipos de usuarios:

Trabajadores de las residencias: Organizan y supervisan las actividades para garantizar su éxito.

Adultos mayores: Participan en las actividades para mantenerse activos y conectados.

Familiares: Pueden seguir las actividades y ver fotos para mantenerse informados.

Voluntarios: Colaboran activamente en las actividades, aportando su tiempo y habilidades.

Desarrollo

Durante el desarrollo de CuidamosJuntos, se siguieron las mejores prácticas en programación, incluyendo metodologías ágiles, diseño centrado en el usuario y pruebas iterativas para asegurar un producto final funcional y accesible.

Entre las tecnologías utilizadas se incluyen:

  • Framework Android para el desarrollo de la app.
  • Firebase para el almacenamiento de datos y autenticación.
  • Material Design para garantizar una interfaz intuitiva y moderna.
